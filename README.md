# Astu STEM

This repository contains two separate applications:

- **Backend**: Node.js + Express + TypeScript API server in the `backend` folder.
- **Frontend**: Next.js + React application in the `frontend` folder.

## Backend

The backend now lives in the `backend` directory.

```bash
cd backend
npm install
npm run dev
# or
npm run build
npm start
```

See `backend/README.md` for more details and environment variable configuration.

## Frontend

The frontend lives in the `frontend` directory.

```bash
cd frontend
npm install
npm run dev
```

Configure the frontend API base URL to point to the backend server (for example `http://localhost:4000`).
