# Astu STEM Complaint Management Backend

Node.js + Express + TypeScript backend with MongoDB for complaint management, authentication, analytics, notifications, and AI chatbot (Gemini + Voyage embeddings).

## Getting Started

1. Change into the backend folder:
   ```bash
   cd backend
   ```
2. Copy `.env.example` to `.env` and fill in values.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run in development:
   ```bash
   npm run dev
   ```
5. Build and start:
   ```bash
   npm run build
   npm start
   ```

Main APIs are under `/api/*` following the structure you described (auth, complaints, categories, users, analytics, notifications, chatbot, files).
